# Air Quality Index (AQI) Analysis 🌍📊

## 📌 Project Overview
This project focuses on analyzing Air Quality Index (AQI) data to understand pollution levels, trends, and key environmental factors affecting air quality.

## 🎯 Objective
To explore AQI data and derive meaningful insights about pollution patterns across different locations and time periods.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Dataset
Air Quality Index dataset (AQI data containing pollutants like PM2.5, PM10, NO2, CO, etc.)

## 🔍 Key Analysis Performed

### 1. AQI Distribution
- Analyzed how AQI values are distributed.
- Identified pollution severity levels.

### 2. Pollutant Analysis
- Studied major pollutants like PM2.5 and PM10.
- Found which pollutants contribute most to poor air quality.

### 3. City-wise AQI Comparison
- Compared AQI levels across different cities.
- Identified highly polluted regions.

### 4. Trend Analysis
- Observed AQI changes over time.
- Identified seasonal pollution patterns.

## 📈 Key Insights
- PM2.5 is a major contributor to high AQI levels.
- Urban areas show higher pollution compared to rural areas.
- AQI levels tend to rise during specific seasons.
- Certain cities consistently experience poor air quality.

## 🚀 Conclusion
This project demonstrates data cleaning, visualization, and analytical skills by extracting meaningful insights from environmental data.
